Static professional portfolio for Nguyen Thanh Dat, positioned for international Junior Frontend, Shopify, WordPress, ecommerce, SaaS landing page, responsive frontend, and website collaboration opportunities.

Live site: <https://liam4503.github.io/shopify-frontend-portfolio/>

## Positioning

- Junior Frontend / Shopify Developer with an IT Support background.

## Run Locally

No installation is required.

1. Open the project folder.
2. Open `index.html` directly in a browser.
3. Optional: use a local static server or VS Code Live Server for easier testing.
