# Nguyen Thanh Dat Portfolio

Static portfolio for Nguyen Thanh Dat, positioned for international Junior Frontend, Shopify, WordPress, ecommerce, SaaS landing page, responsive frontend, and website collaboration opportunities.

Live demo: <https://liam4503.github.io/shopify-frontend-portfolio/>

## Positioning

I am a Junior Frontend / Shopify Developer with an IT Support background. This portfolio presents careful static frontend work, practical Shopify and ecommerce page thinking, WordPress support awareness, documentation quality, testing habits, and honest junior-level growth potential.

I am open to junior frontend roles, Shopify / WordPress collaboration, remote small tasks, paid trial tasks, project-based website support, and freelance website collaboration.

## Accessibility / Performance Notes

- One primary `h1` per page.
- Semantic sections, cards, headings, links, buttons, forms, and accordions.
- Visible keyboard focus states.
- Descriptive link text or accessible labels for repeated project links.
- Responsive grids and full-width mobile actions where useful.
- No large external project image assets.
- Lightweight static HTML/CSS/JavaScript suitable for GitHub Pages.

## Run Locally

No installation is required.

1. Open the project folder.
2. Open `index.html` directly in a browser.
3. Optional: use a local static server or VS Code Live Server when testing route behavior.

## GitHub Pages Compatibility
