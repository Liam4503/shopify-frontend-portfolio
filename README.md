## Run Locally

No installation is required.

1. Open the project folder.
2. Open `index.html` directly in a browser.
3. Optional: open the folder in VS Code and use the Live Server extension for auto-refresh.

## Deploy With GitHub Pages

1. Create or use the GitHub repository for this portfolio.
2. Push all project files to the repository root.
3. In GitHub, open repository `Settings`.
4. Go to `Pages`.
5. Set `Source` to `Deploy from a branch`.
6. Set `Branch` to `main` and folder to `/ (root)`.
7. Save and wait for GitHub Pages to publish the site.
